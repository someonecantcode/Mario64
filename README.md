# Mario64
A WebGL port of Super Mario 64.

[Game Link](https://someonecantcode.github.io/Mario64)

## Controls


←↑→ to Move 

  ↓

X - Jump

C - Jab

W - Camera

S - Uncamera

Space - Crouch

## Combinations
* X + C = Jump Kick
* Space + X = Long Jump
* X (3 times) + Arrows (Running) = Tripple Jump
* Run to the side + X = Side Jump
* On wall + X = Wall Jump
* X + C + Running= Dive
*X + Space  = Ground Pound
